## In the Meantime

To contribute to GeoPulse:

1. **Report bugs or request features** - [Open an issue](https://github.com/tess1o/geopulse/issues)
2. **Fork the repository** - [github.com/tess1o/GeoPulse](https://github.com/tess1o/GeoPulse)
3. **Join discussions** - Check existing issues and PRs
4. **Review the code** - Get familiar with the codebase structure

The project follows standard open-source contribution practices. All contributions are welcome!
